# Django Auction Site

A web-based auction application built with Django that allows users to create, bid on, and manage auction listings.

## Features

- User authentication (login, register, logout)
- Create and manage auction listings
- Place bids on active listings
- Add listings to watchlist
- Categorize listings
- Comment on listings
- View closed auctions and winners

## Prerequisites

- Python 3.8+
- pip (Python package manager)
- Django 4.0+
